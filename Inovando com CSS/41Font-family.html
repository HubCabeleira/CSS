Formatação de texto

O atributo font-family possui propriedades que permitem alterar a fonte dos paragráfos.

Texto de apresentação

Font-family, esta propriedade define uma ou mais fontes para o texto. É importante incluir mais de uma fonte, pois se o browser não suportar a primeira tentarpa a próxima.
