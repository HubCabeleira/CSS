Formatação de texto

O atributo text-transform possui propriedades que permitem alterar a estrutura dos paragráfos.

Texto de apresentação

Text-Tranform com esta opção podemos colocar todo o texto em caixa alta(letras maiúsculas), em caixa baixa (minúsculas), ou ainda, capitular a primeira letra de casa palavra.
Este conteúdo foi editado através de um arquivo html externo.
