Capítulo 1: Atributo Id

O atributo id é utilizado para atribuir identificação a um elemento.

Capítulo 1: Atributo Span

O elemento span é neutro não adiciona qualquer tipo de semântica ao documento. No entanto, pode ser usado no Css para adquirir efeito.

Capítulo 1 : Atributo box model

O box model depende de outros, como margin e padding.

O padding pode ser definindo como o "preenchimento" do elemento. Ele não é a distância entre os elementos (margin), mas sim, o que define a distância interna, ou seja o espaço entre a borda e o conteúdo.
